англо-русский словарь + читалка fb2.
* * *
Установка: запуск только в Qt, файлы БД (dictionaryDB.db, MyWords.db) расположить в папке build-untitled1-Desktop-Debug ;

Программа создана для удобства чтения и перевода книг на английском.

Реализовано: 
- нужные для изучения слова сохранять с переводом. Они отображаются для повторения в двух колонках. Каждую из которых можно скрыть (например, колонку перевода для более эффективного повторения слов) и заново отобразить. Выученные слова также можно удалить.
- чтение книг .fb2 формата.

Что требует доработки:
- необходимо сделать автоперевод выделенного слова с отображением функции сохранения этого слова, чтобы не было необходимости "лезть" в словарь за переводом.

База данных словаря примитивна и составляет около 60 тыс слов. Более сложная является платной.
